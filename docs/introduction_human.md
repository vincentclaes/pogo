# Introduction (Human)

pogo is a dataset-agnostic generative BI app built for biologists who want answers without wrestling with schemas or notebooks. You ask a question in plain language, pogo learns the shape of your data at runtime, and it responds with a clear narrative: the SQL it ran, a table preview, a chart suggestion, and a runnable notebook that records every step.

What this means in practice:
- You can drop in a new dataset with zero schema setup.
- The system translates your intent into SQL and visualizations.
- Each answer is explainable and reproducible because it is exported as a sequential Jupyter notebook.

Use it when you want rapid, trustworthy exploration that is easy to share or re-run later.
